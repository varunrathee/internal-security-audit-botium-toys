## 📄 Audit Source Documentation

This folder contains the primary source documents and reference materials
used to support the internal security audit conducted for the fictional
organization Botium Toys.

### Included Documents

- **Botium Toys – Scope, Goals, and Risk Assessment Report**  
  Defines the audit scope, objectives, assets in scope, and the initial
  risk assessment used to guide the audit process.

- **Controls and Compliance Checklist**  
  Checklist used to evaluate the presence and effectiveness of
  administrative, technical, and physical security controls.

- **Control Categories Reference**  
  Reference document outlining control types and categories used to
  classify and assess security controls during the audit.

- **Completed Controls and Compliance Checklist** *(if included)*  
  Final version of the checklist reflecting the audit findings.

### Purpose
These documents provide the evidence base for the audit analysis and
support the findings, risk assessment, and recommendations documented
in the `audit/` folder.

All documents are included strictly for educational and portfolio
purposes. Sensitive or confidential data has been intentionally excluded.
