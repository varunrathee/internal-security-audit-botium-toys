portfolio activity **“Conduct a Security Audit”** for the fictional
organization Botium Toys.

- **assignment-overview.png**  
  Displays the activity overview and audit scenario describing
  Botium Toys and the scope of the internal security audit.

- **audit-instructions-1.png**  
  Shows the initial step-by-step instructions, including access to

- **audit-instructions-2.png**

- **controls-compliance-checklist.png**

All screenshots are sanitized and contain no sensitive,
personal, or confidential information.
