## 🔍 Audit Analysis

This folder contains the core analysis and findings of the
internal security audit conducted for the fictional organization
Botium Toys.

### Contents
- **controls-assessment.md**  
  Evaluation of administrative, technical, and physical security controls

- **compliance-assessment.md**  
  Assessment of compliance with PCI DSS, GDPR, and SOC standards

- **risk-summary.md**  
  Summary of identified risks, overall risk rating, and potential impact

- **recommendations.md**  
  Actionable security recommendations to improve the organization’s
  security posture and compliance readiness

These files collectively represent the outcome of the internal
security audit and demonstrate practical GRC and audit skills.
