# 📜 Compliance Assessment

This assessment evaluates Botium Toys’ compliance with key regulatory and
industry standards based on the findings of the internal security audit.

---

## 💳 PCI DSS (Payment Card Industry Data Security Standard)

| Requirement | Status |
|------|------|
| Restricted access to cardholder data | No |
| Secure storage and processing environment | No |
| Encryption of cardholder data | No |
| Strong password policies | No |

**Status:** ❌ Not PCI DSS compliant

---

## 🇪🇺 GDPR (General Data Protection Regulation)

| Requirement | Status |
|------|------|
| Protection of EU customer data | Partial |
| 72-hour breach notification plan | Yes |
| Data classification and inventory | No |
| Enforced privacy policies | Yes |

**Status:** ⚠️ Partially GDPR compliant

---

## 🧾 SOC (System and Organization Controls)

| Requirement | Status |
|------|------|
| User access controls | No |
| Confidentiality of PII/SPII | No |
| Data integrity controls | Yes |
| Data availability controls | Yes |

**Status:** ❌ Not SOC compliant

---

## 📌 Summary
Botium Toys does not currently meet full compliance requirements for PCI DSS
or SOC standards and only partially satisfies GDPR obligations.
